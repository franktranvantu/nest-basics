# NestJS Courses
Learn everything you need to master NestJS and tackle modern backend applications at any scale.

## Installation

```bash
$ node --version
$ npm install -g @nestjs/cli
$ nest --version
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
